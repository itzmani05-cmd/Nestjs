## Concept

->Starting from normal authentication using Passport.js local authentication (passport-local), 
->Guards (Local Guards and JWT Guards), 
->JWT authentication, 
->Passport JWT Strategy, and 
->OAuth authentication (Google and GitHub)

## Installation Comments

npm install --save @nestjs/passport passport passport-local
npm install --save-dev @types/passport-local

npm install passport-google-oauth20
npm install -D @types/passport-google-oauth20

npm install passport-github2
npm install -D @types/passport-github2
